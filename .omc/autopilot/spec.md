# GoShell 全功能增强规格说明

## 背景
基于已完成的 Rust→Go 重构项目，对标 XShell 8 和 FinalShell 的功能集，补齐缺失功能并保持内存占用远低于两者（目标：<50MB 常驻，对比 FinalShell 的 200-400MB）。

## 核心原则
1. **内存效率优先**：避免 JavaFX/Electron 级别的内存开销，Go+Fyne 原生二进制
2. **差异化**：本地终端 + 串口是 XShell/FinalShell 没有的优势
3. **UX 对齐**：会话分组、搜索、同步输入、快捷命令必须可用
4. **零依赖外挂**：所有功能内置，无需服务端 Agent

## P0 - 核心体验（必须实现）

### 1. 会话分组与搜索
- Session 结构新增 `Group string` 字段
- 欢迎页会话列表按分组显示，分组标题可折叠
- 顶部搜索框，实时过滤名称/host/user
- 右键会话行：连接/编辑/复制/移动到分组/删除
- 会话对话框增加"分组"输入框

### 2. 标签页增强
- 标签页右键菜单：关闭/关闭其他/关闭右侧/复制会话/重命名/设置颜色
- 标签页颜色标记（6 种预设颜色 + 默认）
- 标签页拖拽排序
- Alt+1~9 切换标签
- Ctrl+Shift+T 新建标签
- Ctrl+Tab/Ctrl+Shift+Tab 循环切换

### 3. 终端日志记录
- 每个标签页可开启日志记录
- 自动保存到 `~/.goshell/logs/{session}_{timestamp}.log`
- 日志带时间戳
- 工具栏增加"开始/停止记录"按钮

### 4. 断线自动重连
- SSH 连接断开时自动检测
- 可配置重连次数（默认 3）和间隔（默认 5s）
- 重连期间显示状态
- 重连失败后显示毛玻璃错误页

## P1 - 差异化功能

### 5. 同步输入模式
- 工具栏"同步"按钮切换
- 启用后，输入会广播到所有已连接的 SSH 标签页
- 状态栏显示"同步模式 ON"
- 仅 SSH 类型标签参与同步

### 6. SFTP 双窗格 + 拖拽
- 左侧本地文件列表，右侧远程文件列表
- 终端 cd 时 SFTP 自动跟随
- 支持拖拽上传/下载
- 右键菜单：上传/下载/删除/重命名/chmod
- 路径栏可手动输入

### 7. 远程进程管理器
- 通过 SSH 执行 `ps aux` 获取进程列表
- 表格显示：PID/USER/CPU/MEM/COMMAND
- 可排序、可搜索
- 一键 kill 进程
- 5s 自动刷新

### 8. 配色方案预设
- 内置 10+ 配色方案（Solarized Dark/Light, Dracula, Monokai, One Dark, GitHub, Nord, Gruvbox, Catppuccin, Tokyo Night）
- 终端颜色独立于 UI 主题
- 设置面板可选

## P2 - 锦上添花

### 9. 快捷命令分组
- QuickCmd 增加 Group 字段
- 面板按分组显示，可折叠
- 右键可编辑/删除

### 10. 命令历史
- 每个会话记录命令历史
- ↑/↓ 浏览历史
- Ctrl+R 反向搜索
- 持久化到 `~/.goshell/history/{session}.json`

### 11. Ping/Trace 监控
- 侧边栏新增"网络"分组
- Ping 延迟实时显示
- Trace 路由跳点
- 多目标 ping

### 12. 终端背景图片
- 设置面板可上传背景图
- 动态模糊效果
- 透明度可调

## 技术约束
- 不引入新的重型依赖
- 所有新增 UI 遵循现有 Fyne 模式
- 保持 go build / go vet 通过
- 内存监控：通过 runtime.ReadMemStats 定期检查
